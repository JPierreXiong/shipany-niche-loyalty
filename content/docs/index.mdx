# Getting Started with Glow

Welcome to Glow, the aesthetic loyalty platform for artisan Shopify stores.

## Quick Start

1. Sign up for a Glow account
2. Connect your Shopify store
3. Customize your brand settings
4. Start adding members

## Features

- Beautiful digital membership cards
- Apple Wallet integration
- Automated reward triggers
- Email campaigns
- Analytics dashboard
